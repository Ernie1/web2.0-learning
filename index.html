Game

HTML + CSS + JS

迷宫

正常赢：移动鼠标，从S开始，到E结束，中间不碰墙，赢得游戏，显示“You Win”
碰墙输：从S开始后，到E结束之前，碰墙，墙变红，输，显示“You Lose”
重置结果：离开迷宫，墙恢复正常；从S开始时，隐藏结果显示
发现作弊：如果用户未经过S，就指到E，又或者指向S之后，从迷宫外绕路指向E，显示"Don't cheat, you should start form the 'S' and move to the 'E' inside the maze!"

打地鼠

能够随机出现地鼠，鼠标能够击中（点击正确，地鼠消失，出现新地鼠；点击错误，地鼠不消失）
